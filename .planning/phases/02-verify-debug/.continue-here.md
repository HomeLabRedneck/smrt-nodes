# BLOCKING CONSTRAINTS — Read Before Anything Else

> These are not suggestions. Each constraint below was discovered through failure.
> Acknowledge each one explicitly before proceeding.

- [ ] CONSTRAINT: ZIP-path-separator — Windows `Compress-Archive` produces backslash paths; Godot mod loader requires forward-slash paths. Must use .NET `System.IO.Compression.ZipFile` with explicit `path.Replace('\', '/')` for all packaging steps.
- [ ] CONSTRAINT: update_connections-dispatch — Game engine's `_on_create_connection()` does NOT reliably call our GDScript `update_connections()` override. Must hook `connection_out_set` signal in `_ready()` AND add tick-time fallback `if transfer.is_empty() and not outputs.is_empty(): update_connections()`.

**Do not proceed until all boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Compress-Archive for packaging | Produces backslash ZIP entry paths; mod loader silently fails to find `manifest.json` | blocking | Always use .NET ZipArchive with forward-slash path construction |
| Relying on update_connections() override firing | Game engine dispatch does not reach GDScript override reliably | blocking | connection_out_set signal + tick-time fallback already in place; do not remove |
